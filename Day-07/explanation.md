Calculate Fibonacci number efficiently.

Approach 1 - Iterative (Optimal):
1. Use two variables to track previous two numbers
2. Iterate from 2 to n
3. Update variables in each iteration

Approach 2 - Recursive with Memoization:
1. Cache computed values
2. Avoid redundant calculations

Key Points:
- Iterative is O(n) time, O(1) space
- Recursive is simpler but less efficient
